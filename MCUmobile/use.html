3. Setup MCU on Bluestacks

Before starting, we recommend creating a new calendar for timetable.

3a. Open MCU from the Android tab. Click the menu icon on top-left.

3b. Click "Settings"

3c. Click "Calendar Used for New Events"

3d. Set it to the new calendar you have created

If you cannot find the new calendar, don't worry. Close bluestacks and start it again to synchronize the changes and try again.

3d. Check "Calendar Sync"

3e. Restart MCU for Android. Sit back and relax.

Close the app and reopen it from the Android tab. Wait for an hour or so and the calendar will be automatically synced.

You're done!

Note: Bluestacks will run in the background of your computer.
Please keep it running in the background or else the calendar will not sync properly.
